# Calculator 2.0 Function Coverage

This file is the exported public-function checklist for the 2.0 migration. It
tracks the 1.0 documented scalar surface against the new `:v2` runtime.

Status values:

- `migrated`: handled by the 2.0 `FunctionRegistry` or 2.0 CAS
- `symbolic`: parsed and preserved as an expression, but not numerically
  evaluated by the 2.0 backend yet
- `legacy`: still handled by the 1.0 runtime only

## Arithmetic And Numeric Helpers

| Function | 2.0 status |
| --- | --- |
| `+`, `-`, `*`, `/`, `^` | migrated |
| `pow(a, b)` | migrated for integer exponents |
| `abs(x)` | migrated for real exact values |
| `sign(x)` | migrated for real exact values |
| `step(x)`, `heaviside(x)` | migrated |
| `delta(x)`, `impulse(x)` | migrated |
| `floor(x)`, `ceil(x)`, `round(x)`, `trunc(x)` | migrated |
| `min(...)`, `max(...)`, `clamp(x, a, b)` | migrated |
| `sum(...)`, `mean(...)`, `avg(...)` | migrated |
| `median`, `mode`, `var`, `variance` | migrated |
| `percentile`, `quartile`, `std` | legacy |
| `idiv(a, b)` | migrated for integers |
| `mod(a, b)` | migrated for integers |

## Integer Helpers

| Function | 2.0 status |
| --- | --- |
| `gcd(...)`, `lcm(...)` | migrated |
| `factorial(n)` | migrated |
| `nCr(n, r)`, `binom(n, r)`, `nPr(n, r)` | migrated |
| `fib`, `fibonacci`, `is_prime`, `next_prime`, `prev_prime`, `prime_pi`, `euler_phi`, `phi` | migrated for small exact integers |
| `mobius` | legacy |

## Elementary And Special Functions

| Function | 2.0 status |
| --- | --- |
| `sqrt(x)` | migrated for exact perfect integer/rational squares |
| `exp`, `ln`, `log`, `sin`, `cos`, `tan` | migrated for exact known values; symbolic / CAS rules otherwise |
| `sec`, `csc`, `cot` | symbolic / CAS derivative and integration rules |
| `sinh`, `cosh`, `tanh`, `asin`, `atan`, `asinh`, `atanh` | migrated for exact known values; symbolic / CAS rules otherwise |
| other inverse trig and hyperbolic functions | symbolic / selected derivative rules |
| `gamma`, `beta`, `zeta`, `erf`, `erfc`, `bessel` | legacy |
| `pi`, `e` | symbolic constants in 2.0 |
| `i` | migrated numeric complex constant |

## Complex Helpers

| Function | 2.0 status |
| --- | --- |
| `real(z)`, `imag(z)`, `conj(z)` | migrated |
| `complex(re, im)`, `abs(z)`, `arg(z)`, `polar(r, theta)` | legacy |

## CAS

| Function | 2.0 status |
| --- | --- |
| `simplify(expr)` | migrated for current 2.0 expression forms |
| `diff(expr, x)` | migrated for current 2.0 expression forms |
| `gradient(expr, [vars...])` | migrated |
| `jacobian([exprs...], [vars...])` | migrated |
| `hessian(expr, [vars...])` | migrated |
| `integrate(expr, x)` | migrated for Phase 5 table, polynomial, selected rational, substitution, integration-by-parts, and special-function rules |
| unevaluated `integral(expr, x)` output | migrated |
| transforms, series, limits | legacy |

## Units

| Function | 2.0 status |
| --- | --- |
| `celsius`, `f2c`, `fahrenheit`, `c2f`, `kelvin` | migrated |

## Matrix, Analysis, Script, Probability

These remain on the 1.0 runtime until later migration phases.
