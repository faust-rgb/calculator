## 简介

这是一个功能强大的 C++ 命令行计算器，支持：

- 基本和高级数学运算
- 矩阵和向量运算
- 符号微积分（求导、积分、极限、泰勒展开）
- 常微分方程数值求解
- 脚本编程
- 精确分数模式
- 符号常数显示

所有数学函数均从零实现，不依赖标准数学库 `<cmath>`。

## 快速开始

### 编译

```bash
make
```

### 运行

```bash
./calculator
```

### 简单示例

```
> 1 + 2 * 3
7
> sin(pi / 2)
1
> sqrt(2) * sqrt(2)
2
> exit
```

## 基本运算

### 算术运算符

| 运算符 | 说明 | 示例 |
|--------|------|------|
| `+` | 加法 | `2 + 3` |
| `-` | 减法 | `5 - 2` |
| `*` | 乘法 | `3 * 4` |
| `/` | 除法 | `7 / 3` |
| `^` | 幂运算 | `2 ^ 10` |
| `-` | 一元负号 | `-5` |
| `+` | 一元正号 | `+3` |

### 优先级

1. 括号 `()`
2. 幂运算 `^`（右结合）
3. 一元正负号
4. 乘法、除法 `* /`
5. 加法、减法 `+ -`

注意：`-2^2` 被解析为 `-(2^2) = -4`，而非 `(-2)^2 = 4`。

### 前缀整数字面量

- `0b` 二进制：`0b1010` = 10
- `0o` 八进制：`0o77` = 63
- `0x` 十六进制：`0xFF` = 255

示例：
```
> 0b1010 + 0xF
25
> 0o77
63
```

### 泰勒展开与级数

```
> taylor(f, a, n)
```

在点 `a` 处展开 `n` 阶泰勒多项式。

示例：
```
> taylor(sin(x), 0, 5)
x - x^3/6 + x^5/120
```

### Pade 近似

```
> pade(expr, m, n)
> pade(expr, a, m, n)
```

构建 Pade 有理近似（从泰勒系数）。

### Puiseux 级数

```
> puiseux(expr, degree, denominator)
> puiseux(expr, a, degree, denominator)
```

使用分母网格展开 Puiseux 级数。

### 级数求和

```
> series_sum(expr, n, lower, upper)
> summation(expr, n, lower, upper)
```

计算级数和。支持多项式被加数（最高3次）和几何级数。使用 `inf`、`oo` 或 `infinity` 作为上界计算无穷几何级数。

### 符号简化

```
> simplify(expr)
```

简化符号表达式。

示例：
```
> simplify(x*x/x)
x
> simplify((x + 1) + 2 * (1 + x))
3 * (x + 1)
```

### 常微分方程

#### 一阶 ODE

```
> ode(rhs, x0, y0, x1)
> ode(rhs, x0, y0, x1, steps)
> ode(rhs, x0, y0, x1, steps, params_vec)
> ode(rhs, x0, y0, x1, steps, event_expr)
> ode(rhs, x0, y0, x1, steps, event_expr, params_vec)
> ode_table(rhs, x0, y0, x1)
> ode_table(rhs, x0, y0, x1, steps)
> ode_table(rhs, x0, y0, x1, steps, params_vec)
> ode_table(rhs, x0, y0, x1, steps, event_expr)
> ode_table(rhs, x0, y0, x1, steps, event_expr, params_vec)
```

求解初值问题 `y' = rhs(x, y)`，其中 `y(x0) = y0`。

- `ode` 返回 `x1` 处的近似值
- `ode_table` 返回采样点的表格
- 可选参数 `params_vec` 传递参数向量，在表达式中可用 `p`, `p1`, `p2`, ... 访问
- 可选参数 `event_expr` 是标量表达式，当事件穿过零时停止积分

使用经典 RK4 方法，默认 100 步（`ode_table` 默认 10 步）。

#### ODE 系统

```
> ode_system(rhs_vec, x0, y0_vec, x1)
> ode_system(rhs_vec, x0, y0_vec, x1, steps)
> ode_system(rhs_vec, x0, y0_vec, x1, steps, params_vec)
> ode_system(rhs_vec, x0, y0_vec, x1, steps, event_expr)
> ode_system(rhs_vec, x0, y0_vec, x1, steps, event_expr, params_vec)
> ode_system_table(rhs_vec, x0, y0_vec, x1)
> ode_system_table(rhs_vec, x0, y0_vec, x1, steps)
> ode_system_table(rhs_vec, x0, y0_vec, x1, steps, params_vec)
> ode_system_table(rhs_vec, x0, y0_vec, x1, steps, event_expr)
> ode_system_table(rhs_vec, x0, y0_vec, x1, steps, event_expr, params_vec)
```

求解非线性 ODE 系统，右端项求值为向量。

- 在 `ode_system` 内部，状态可用 `y1`, `y2`, ... 访问，也可用向量变量 `y` 访问
- `ode_system_table` 返回列格式为 `[x, y1, y2, ...]` 的矩阵

### 求根

| 函数 | 说明 |
|------|------|
| `solve(expr, x0)` | 求方程 expr = 0 的根（从 x0 开始迭代）|
| `bisect(expr, a, b)` | 二分法求根 |
| `secant(expr, x0, x1)` | 割线法求根 |
| `fixed_point(expr, x0)` | 不动点迭代 |

| 函数 | 说明 |

| `lp_max(c, A, b, lower, upper)` / `lp_max(c, A, b, Aeq, beq, lower, upper)` | 线性规划最大化 |
| `lp_min(c, A, b, lower, upper)` / `lp_min(c, A, b, Aeq, beq, lower, upper)` | 线性规划最小化 |

## 已知限制

### 解析限制

- 支持科学计数法，例如 `1e-3`
- 表达式导向的解析器，不是完整的符号代数解析器

### 精确模式限制

- 仅限于有理数运算和相关精确友好操作
- 无法保持有理数的表达式会回退到小数显示
- 不是完整的符号 CAS
- 符号积分已覆盖更多有理函数和三角恒等式形式，但仍不是完整 Risch 判定系统
- `critical` 会对 1-3 维孤立驻点输出 Hessian 分类；非线性驻点搜索仍是有界启发式搜索，不保证全局完备

### 进制转换限制

- `bin`、`oct`、`hex`、`base` 只接受整数
- `base(n, b)` 支持 2-16 进制

### 位运算限制

- 位运算与程序员辅助函数只接受整数
- `rol` / `ror` / `reverse_bits` 使用 64 位二进制补码语义

- 位运算函数只接受整数
- 移位计数必须非负

### 数值模型

- 内部使用 `double` 进行小数计算
- 不支持任意精度
- 某些操作本质上是近似的，依赖容差处理
- 接近零的值会规范化为 `0` 以提高可读性
- `fft` / `ifft` 在长度为 2 的幂时使用快速傅里叶变换；其他长度回退到直接 DFT
- 符号表达式使用会话内节点驻留、结构键缓存和 LRU 缓存来减少重复分配与重复化简

### 矩阵限制

- `eigvals` 和 `eigvecs` 目前仅支持实特征值
- `qr_q`、`qr_r`、`lu_l`、`lu_u` 目前需要方阵
- `lu_l` 和 `lu_u` 使用无选主元的 LU 分解

### 持久化限制

- 保存/加载格式是简单文本，无版本控制
- 加载无效文件会引发错误，而不是尝试部分恢复
- 矩阵变量暂不支持保存/加载
